# Developer Portfolio — Systems Automation & Digital Commerce

A high-performance, single-page developer portfolio built using **Next.js (App Router)**, **React**, and **Tailwind CSS**. Designed for extreme performance, zero layout shift, and clean typography.

## Core Features

- **Hero Section**: Focused headline highlighting systems automation and e-commerce engineering, with a direct call-to-action linking to your GitHub profile.
- **Case Study Section**: Focuses on the *Eldorado Digital Storefront*, detailing database sync, automatic dispatch webhooks, and latency metrics.
- **Simulated Sales Dashboard**: A custom-designed CSS dashboard representation showing live fulfillment logs, active transaction syncing, and a transaction velocity vector chart.
- **Technical Projects Grid**: Responsive cards with minimalist iconography for:
  - *Automated Fulfillment Engine* (Python, Async, API)
  - *Hardware Execution Scripts* (Raspberry Pi Pico, Python)
  - *Local AI Orchestration* (Gemini API, Ollama, Python)
- **Skills Stack**: Grid of core skills styled with modern borders and micro-hover states.
- **Strict Dark Theme**: Deep zinc palette with emerald green accents enforced globally in CSS.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build and Deployment

To compile the application into an optimized static production build:
```bash
npm run build
```
The build runs typechecking and ESLint, generating a zero-warnings compile optimized for fast load times.
